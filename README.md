# Derco - Test
Test de servicio de consumo API y visualización de datos

## Instalación
```sh
git clone https://github.com/oscardcs/derco.git
yarn
yarn start
```
- Nota: El puerto por defecto del servidor será el puerto 3000
- Realizado con react 17.0.2
